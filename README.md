# Web Design Sample

This project is a client-facing website designed and implemented as a portfolio sample.

## Goals
- Clear structure and hierarchy
- CMS-ready layout
- Responsive behavior
- Easy content updates for non-technical users

## Design
- Desktop-first (1440px)
- Mobile adaptation (375px)
- Simple color palette
- Max 2 font families
- Components designed for reuse

## Implementation
- HTML5 + CSS3
- No frameworks
- Responsive grid and spacing
- Maintainable structure
